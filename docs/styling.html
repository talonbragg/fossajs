Styling

Fossa gives a styling object constructor sto easily style elements without having to load javascript and css at the same time.

Here is an example:

        
          var style = new style({
            el: 'styledElement',
            style: padding: 2;margin:20;
          });

First el is the id of the element that you want to style. Then, you are probably wondering, why isn't the style element a string? This is because the javascript changes it to a string once you load the html DOM.
